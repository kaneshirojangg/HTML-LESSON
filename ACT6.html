ACT 6-B: Definition of Terms

ROM (Read-Only Memory):
Permanent memory in a computer that stores critical data and instructions,
like firmware, which can't be modified easily.

HTTPS (HyperText Transfer Protocol Secure):
A secure version of HTTP that encrypts data exchanged between a web browser
and a server to protect sensitive information.

CPU (Central Processing Unit):
The "brain" of the computer that executes instructions and
performs calculations necessary for running programs.

RAM (Random Access Memory):
Temporary memory that stores data being used by the computer,
which is lost when the power is turned off.

BIOS (Basic Input/Output System):
Firmware that initializes hardware when you power on the computer,
and hands control over to the operating system.

GPU (Graphics Processing Unit):
A specialized processor designed to handle rendering graphics and images,
also used for other parallel processing tasks.

IP Address:
A unique identifier for devices on a network,
allowing them to communicate with each other over the internet.

DNS (Domain Name System):
A system that translates human-friendly domain names
(e.g., www.example.com) into IP addresses computers use to locate resources.

SSD (Solid State Drive):
A faster, more durable storage device compared to traditional hard drives,
as it uses flash memory with no moving parts.

Firewall:
A security system that controls incoming and outgoing network
traffic based on rules to protect against unauthorized access.
